IP地址的获取

发表于 2019-09-10 分类于 linux ， dhcp

在Linux最小安装之后，一般需要手动配置网络地址。

一、简单的利用DHCP获取

1、进网卡配置文件

cd /etc/sysconfig/network-scripts/

2、列出network-scripts目录下的文件

ls /etc/sysconfig/network-scripts/

3、找到ifcfg-ens33,进入ifcfg-ens33文件，修改onboot的no为yes

4、重启网络地址：

5.重启或关机后再次连接

二、手动配置IP。

1、进入网卡配置文件

cd /etc/sysconfig/network-scripts/

2、ls列出network-scripts目录下的文件

ls /etc/sysconfig/network-scripts/

3、找到ifcfg-ensXX,比如ifcfg-ens33,配置ifcfg-ens33文件

在第三步中，得到子网和掩码，所用IP可以从192.168.59.3-192.168.59.254中随便获取一个。

4、保存退出，并且重启网络地址。

Systemctl restart network

5.重启或关机后再次连接

如果IP是自己配置的，则说明网络配置成功。

ip -a / ifconfig #查看网卡
